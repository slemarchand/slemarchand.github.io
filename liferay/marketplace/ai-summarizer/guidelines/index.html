AI Summarizer for Liferay

The AI Summarizer for Liferay enhances the Document Library by providing AI-generated summaries for documents with a single click. Once installed, a new “Summarize” button becomes available in the document detail screen, allowing users to request a summary in the background and retrieve it moments later via a popup.

Key Features

  • One-click document summarization directly inside the Document Library
  • Background processing for optimal user experience
  • AI-agnostic integration: choose your AI provider
  • Supports SaaS, PaaS, and on-premise Liferay deployments

Included Liferay Client Extensions

This app is delivered as a set of Liferay Client Extensions:

  • slemarchand-aisummarizer-batch.zip
  • slemarchand-aisummarizer-custom-element.zip
  • slemarchand-aisummarizer-etc-service.zip
  • slemarchand-aisummarizer-global-js.zip
Important: The following Feature Flags must be enabled:
  • LPD-30371 – Global JavaScript Client Extensions
  • LPD-48862 – OAuth2 JavaScript Client

Supported AI Providers

You can choose your preferred AI provider at configuration time:

  • OpenAI
  • Google Gemini

Installation

  1. Download the application package from the Liferay Marketplace.
  2. Navigate to Control Panel → Apps → App Manager.
  3. Upload and deploy the provided Client Extensions or install via Marketplace deployment.
  4. Ensure the required Feature Flags are enabled (see above).

Configuration

Once installed, configure the app:

  1. Go to Control Panel → AI Summarizer Settings.
  2. Select your desired AI provider (OpenAI or Google Gemini).
  3. Provide the appropriate API Key and configuration values.
  4. Save the configuration.

How to Use

  1. Open the Document Library and navigate to a document.
  2. Click on the document to view its detail page.
  3. Click the “Summarize” button.
  4. The summary will be generated in the background.
  5. Once ready, the button changes to “Show Summary”.
  6. Click it to display the summary in a pop-up.
✅ Tip: Summaries may vary depending on the AI model and the document type. You may switch AI providers anytime through the app configuration.

Compatibility

This application is compatible with:

  • Liferay SaaS
  • Liferay PaaS
  • Liferay Self-Hosted / On-Premise
